# Online Chess WebSocket

## Situação
Desenvolvimento de um backend para um jogo de xadrez online capaz de permitir partidas entre dois jogadores com **sincronização de jogadas em tempo real**.

## Tarefa
Projetar e implementar um servidor responsável por gerenciar partidas de xadrez, controlar o estado do jogo e validar todas as regras da partida, além de possibilitar a comunicação entre jogadores conectados.

## Ação
Foi desenvolvido um backend utilizando **Java e Spring Boot**, com comunicação bidirecional em tempo real através de **Spring WebSocket**.  
A lógica completa do jogo de xadrez foi implementada aplicando princípios de **Programação Orientada a Objetos**, incluindo modelagem das peças e regras do jogo.  
Também foi criado um sistema de **matchmaking** para pareamento automático de jogadores e gerenciamento de partidas online e amistosas.

## Resultado
O sistema permite a criação e gerenciamento de partidas multiplayer de xadrez, com validação completa das regras do jogo e atualização das jogadas entre os jogadores em **tempo real**.

## Tecnologias utilizadas

- Java 17  
- Spring Boot  
- Spring WebSocket  
- Maven
